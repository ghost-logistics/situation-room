### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. **Clone or Download**
   ```bash
   git clone https://github.com/YOUR-USERNAME/star-sentinel.git
   cd star-sentinel
   ```

2. **Deploy to Web Hosting**
   - **GitHub Pages:** Upload to repository, enable Pages
   - **Netlify:** Drag folder to app.netlify.com
   - **Vercel:** `vercel deploy`
   - **Or open locally:** Just open `index.html` in your browser!

3. **Start Tracking!**
   - Open your deployed URL (or local file)
   - Wait 2-3 seconds for satellites to load
   - Interact with the 3D globe
