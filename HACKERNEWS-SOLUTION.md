## What Changed:

### API Endpoint:
```
https://hacker-news.firebaseio.com/v0/topstories.json
```

### How It Works:
1. Fetches top 100 stories from HackerNews
2. Filters by keywords for each thread:
   - **Military Space**: space, satellite, orbital, missile, rocket, defense, military, pentagon
   - **Middle East**: israel, middle east, gaza, iran, idf, hezbollah, hamas, palestine, syria, lebanon
   - **Space Science**: nasa, spacex, astronomy, astrophysics, telescope, mars, moon, solar, cosmic
3. Displays top 15 matching stories per thread
4. Updates every 5 minutes

### Data Displayed:
- ✅ Title (full text)
- ✅ Source (HackerNews)
- ✅ Author
- ✅ Upvotes (accurate)
- ✅ Comments (accurate)
- ✅ Time posted
- ✅ Clickable link to story
