# 🚀 Quick Deploy - 60 Seconds

## Fastest: Netlify Drop

1. Go to https://app.netlify.com/drop
2. Drag the entire `situation-room-website` folder
3. Done! Get your URL instantly

## Fast: Vercel CLI

```bash
npm i -g vercel
vercel
```
Follow prompts. Done in 30 seconds.

## Fast: GitHub Pages

```bash
# In the website folder
git init
git add .
git commit -m "Deploy"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git
git push -u origin main
```

Then: GitHub repo → Settings → Pages → Enable

URL: `https://YOUR_USERNAME.github.io/REPO_NAME`

## Local Testing

```bash
npm start
```

Opens at http://localhost:8080

---

## That's It!

No build step. No configuration. Just pure HTML/CSS/JS.

Choose your platform and go! 🎉
